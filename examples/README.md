
# LibCommand Examples

Create commands with typed arguments, constraints and subcommands instead of manually parsing raw strings.

## Minimal integration checklist

1. Add the library as a dependency or install the standalone PHAR.
2. Check `docs/compatibility.md` for supported PHP/PMMP versions.
3. Keep runtime registration inside your plugin enable/bootstrap flow.
4. Validate behavior on a local PocketMine-MP server before production use.
