
# Support

Use GitHub issues for reproducible bugs and feature requests for `LibCommand`.

Before opening an issue, please check:

- the README
- the `docs/` directory
- existing open and closed issues
- the latest release notes or changelog

For support, include:

- `LibCommand` version
- PocketMine-MP version, when applicable
- PHP version
- full error/crashdump, when applicable
- minimal code/config needed to reproduce the issue

Security-sensitive reports must follow `SECURITY.md` instead of public issues.
