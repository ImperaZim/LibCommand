
# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| 2.x | Yes |
| Older versions | Limited / best effort |

## Reporting a vulnerability

Please do not open a public issue for security-sensitive reports.

Send details privately to `imperazim2@gmail.com` with:

- Repository: `LibCommand`
- Affected version
- PocketMine-MP version, when applicable
- PHP version
- Steps to reproduce
- Impact and possible mitigation, if known

Reports are reviewed on a best-effort basis. If the issue is confirmed, a fix or mitigation note should be published before or together with public disclosure.
